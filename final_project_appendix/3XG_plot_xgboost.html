R Markdown

This is an R Markdown document. Markdown is a simple formatting syntax for authoring HTML, PDF, and MS Word documents. For more details on using R Markdown see http://rmarkdown.rstudio.com.

When you click the Knit button a document will be generated that includes both content as well as the output of any embedded R code chunks within the document. You can embed an R code chunk like this:

library(pROC)
## Type 'citation("pROC")' for a citation.
## 
## Attaching package: 'pROC'
## The following objects are masked from 'package:stats':
## 
##     cov, smooth, var
roc_ses <- readRDS("roc_ses.rds")
roc_svi <- readRDS("roc_svi.rds")

Including Plots

You can also embed plots, for example:

par(mfrow = c(2, 3))

plot_roc_panel(roc_ses$train$linear,       NULL, "Train: Linear (SES)")
plot_roc_panel(roc_ses$train$nonlinear,    NULL, "Train: Nonlinear (SES)")
plot_roc_panel(roc_ses$train$nonlinear_fs, NULL, "Train: Nonlinear + FS (SES)")

plot_roc_panel(roc_ses$test$linear,       NULL, "Test: Linear (SES)")
plot_roc_panel(roc_ses$test$nonlinear,    NULL, "Test: Nonlinear (SES)")
plot_roc_panel(roc_ses$test$nonlinear_fs, NULL, "Test: Nonlinear + FS (SES)")

par(mfrow = c(2, 2))

plot_roc_panel(roc_svi$train$linear,    NULL, "Train: Linear (SVI)")
plot_roc_panel(roc_svi$train$nonlinear, NULL, "Train: Nonlinear (SVI)")

plot_roc_panel(roc_svi$test$linear,    NULL, "Test: Linear (SVI)")
plot_roc_panel(roc_svi$test$nonlinear, NULL, "Test: Nonlinear (SVI)")

Note that the echo = FALSE parameter was added to the code chunk to prevent printing of the R code that generated the plot.
